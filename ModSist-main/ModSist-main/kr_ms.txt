*Модуль 0: Область описания*
T_w QTABLE ch_1,0.01,0.01,100 ; плотность распределения времени ожидания в очереди 
T_u TABLE M1,25,100,100 ; время пребывания в сети

* Модуль 1: моделирование процессов поступления и обслуживания заявок
GENERATE (Exponential(1, 0, 0.069)) ; входное время
QUEUE ch_1 ; регистрация поступления заявки в очередь
Seize 1 ; попытка занять один из приборов узла
DEPART ch_1 ; регистрация покидания заявки очереди
ADVANCE (Exponential(2, 0, 0.02)) ; время обслуживания
RELEASE 1 ; выход обслуженной заявки из узла
TABULATE T_u ; запись данных в таблицу
TERMINATE 1 ; удаление из модели обслуженной заявки
START 10000 ; запуск модели
